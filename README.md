# Ember Kitchen — Cinematic Restaurant Delivery App

A visually stunning restaurant delivery web app with cinematic animations and a premium user experience. Built with Next.js 14, Supabase, and Framer Motion.

## Live Demo
[emberkitchen.vercel.app](https://emberkitchen.vercel.app)

## Features
- Cinematic hero sections with smooth scroll animations
- Restaurant and menu browsing with category filters
- Cart and checkout flow
- Order tracking and management
- User authentication and profile
- Fully responsive mobile-first design

## Tech Stack
- **Frontend:** Next.js 14, TypeScript, Tailwind CSS, Framer Motion
- **Backend:** Supabase (PostgreSQL + Auth)
- **Deployment:** Vercel

## Getting Started
```bash
npm install
cp .env.example .env.local
npm run dev
```
